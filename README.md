## 📖 Overview

**English Janala** is a clean, responsive static website for interactive English language learning. It presents vocabulary, grammar, and learning resources in a structured, user-friendly layout — built with HTML, CSS, and Tailwind CSS.

> **Live at** → [https://imoloy.github.io/EnglishJanala](https://imoloy.github.io/EnglishJanala)

---

## 🛠️ Technologies Used

| Technology | Purpose |
|---|---|
| **HTML5** | Semantic page structure |
| **CSS3** | Custom styling and layout |
| **Tailwind CSS** | Utility-first responsive design |
| **JavaScript** | Interactive UI behavior |
| **Google Fonts** | Hind Siliguri & Poppins typography |

---

## ✨ Core Features

- **Vocabulary Sections** — Organized word lists with pronunciation guides
- **Grammar Lessons** — Clean layout for structured grammar content
- **Bilingual Support** — Bengali-friendly font (Hind Siliguri) alongside English
- **Responsive Layout** — Works seamlessly on mobile, tablet, and desktop
- **Fast & Lightweight** — Zero npm dependencies, pure HTML/CSS/JS

---

## 📦 Dependencies

No npm packages. External CDN resources:

| Resource | Source |
|---|---|
| Tailwind CSS | CDN |
| Google Fonts (Hind Siliguri, Poppins) | `fonts.googleapis.com` |

## 🚀 Run Locally

1. **Clone the repository**

   ```bash
   git clone https://github.com/iMoloy/EnglishJanala.git
   cd EnglishJanala
   ```

2. **Open in browser**

   ```bash
   open index.html
   # or serve with
   npx serve .
   ```

No build step required.
